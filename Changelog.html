Requisitos:

É necessário instalar o JAVA JRE 1.5 para rodar.

O JDK 1.5 é necessário para compilar o jogo se necessário.

Se desejar baixar: http://java.sun.com/

Instruções:

O Usuário deverá fornecer ao jogo:

1. Nome do arquivo que contenha a fase e o caminho que o documento se encontra

O Jogo irá iniciar com o player 1.
O dano de cada tiro será informado no arquivo da pasta, o jogador terá tempo para atirar.

Controles :

O Usuário só poderá jogar utilizando o teclado, com as teclas “Left” e “Rigth” o player irá se mover, com a tecla “Up” e “Down” ele controlará o angulo. Com a barra de espaço irá se medir a força que o usuário desejar e com a tecla “Control” fará o player pular.
